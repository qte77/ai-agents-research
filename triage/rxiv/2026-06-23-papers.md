## Extracts

### MemoryWAM: Efficient World Action Modeling with Persistent Memory

- arxiv: <https://arxiv.org/abs/2606.20562>
- summary: MemoryWAM is a world action model designed for robust robotic manipulation that efficiently integrates memory and dynamics modeling to improve performance in non-Markovian environments.
- methods: hybrid memory design, tailored attention mechanism
- findings: MemoryWAM outperforms strong VLA and WAM baselines; MemoryWAM maintains favorable computational efficiency; MemoryWAM supports memory-dependent decision-making with reduced inference latency and GPU memory usage

### Execution-State Capsules: Graph-Bound Execution-State Checkpoint and Restore for Low-Latency, Small-Batch, On-Device Physical-AI Serving

- arxiv: <https://arxiv.org/abs/2606.20537>
- summary: This study introduces execution-state capsules for low-latency, small-batch on-device physical-AI serving, enhancing responsiveness in interactive LLM agents and other systems.
- methods: graph-bound checkpoint and restore mechanism, snapshot and restore, forking and rolling back execution state
- findings: Capsule restore is byte-exact at the stored-state level and token-identical under greedy decode.; GPU-resident snapshot and restore are sub-millisecond.; TTFT speedup over cold prefill grows from 3.9x at 2k tokens to 27x at 16k tokens.; Recurrent state is load-bearing in contrast to KV-only ablation.

### LedgerAgent: Structured State for Policy-Adherent Tool-Calling Agents

- arxiv: <https://arxiv.org/abs/2606.20529>
- summary: The study introduces LedgerAgent, a method for tool-calling agents that maintains task states in a separate ledger to improve policy adherence and decision-making in customer service domains.
- methods: inference-time method, state management, ledger system
- findings: LedgerAgent improves average pass@k over standard prompt-based approaches; Largest gains observed under stricter multi-trial consistency metrics

### HumanScale: Egocentric Human Video Can Outperform Real-Robot Data for Embodied Pretraining

- arxiv: <https://arxiv.org/abs/2606.20521>
- summary: This study compares egocentric human video and teleoperated real-robot trajectories as pretraining data sources for embodied foundation models, revealing that egocentric data can lead to superior performance.
- methods: systematic study, filtering and labeling pipeline
- findings: Egocentric data can lead to superior performance compared to teleoperated real-robot data.; Models pretrained on egocentric data achieve a 24% lower validation loss on real-robot action prediction.; Models pretrained on egocentric data have 52.5% and 90% higher success rates on in-distribution and out-of-distribution real-robot task execution, respectively.

### Sovereign Execution Brokers: Enforcing Certificate-Bound Authority in Agentic Control Planes

- arxiv: <https://arxiv.org/abs/2606.20520>
- summary: This paper introduces the Sovereign Execution Broker (SEB), a runtime enforcement boundary for managing certified authority in autonomous agents connected to cloud infrastructures.
- methods: runtime enforcement, certificate verification, policy checking, identity minting, API invocation, decision recording
- findings: SEB provides a mandatory enforcement point for certified authority; SEB separates proposal, admission, and execution; The prototype implementation shows measurable latency overheads and security under fault injection

### Multi-LCB: Extending LiveCodeBench to Multiple Programming Languages

- arxiv: <https://arxiv.org/abs/2606.20517>
- summary: Multi-LCB is introduced as a new benchmark for evaluating large language models across twelve programming languages, addressing the limitations of LiveCodeBench (LCB) which is restricted to Python.
- methods: transforming Python tasks, contamination controls, evaluation protocol
- findings: evidence of Python overfitting; language-specific contamination; substantial disparities in multilingual performance

### S-Agent: Spatial Tool-Use Elicits Reasoning for Spatial Intelligence

- arxiv: <https://arxiv.org/abs/2606.20515>
- summary: The study introduces S-Agent, a spatial tool-use agentic paradigm that enhances spatial reasoning over continuous multi-view images and videos by integrating evidence accumulation and temporal memory mechanisms.
- methods: spatio-temporal evidence accumulation, semantic planning, temporal memory mechanism, supervised fine-tuning
- findings: S-Agent improves both open-source and closed-source VLMs in a training-free manner.; S-Agent-8B significantly surpasses similar-scale baselines and performs comparably to advanced closed-source models.

### Probe-and-Refine Tuning of Repository Guidance for Coding Agents

- arxiv: <https://arxiv.org/abs/2606.20512>
- summary: This paper introduces probe-and-refine tuning, a method for improving LLM-based coding agents' guidance by using synthetic bug-fix probes to enhance repository context, demonstrating significant performance improvements over static knowledge bases and unguided approaches.
- methods: probe-and-refine tuning, synthetic bug-fix probes, single-shot LLM calls
- findings: Probe-and-refine achieves a 33.0% mean resolve rate compared to 28.3% for static knowledge base and 25.5% for unguided baseline.; Refined guidance produces evaluable patches for 14.5 percentage points more instances while maintaining constant per-patch precision (~59%).; Guidance allows agents to use a larger step budget productively.

### Efficient and Sound Probabilistic Verification for AI Agents

- arxiv: <https://arxiv.org/abs/2606.20510>
- summary: This study introduces a framework for verifying security policies in AI agents operating in complex environments, addressing the limitations of existing deterministic approaches by incorporating probabilistic predicates.
- methods: runtime monitoring, distributionally robust optimization
- findings: Outperforms prior art; Improves security-utility trade-off; Ensures rigorous bounds on the probability of policy violation

### What Do Safety-Aligned LLMs Learn From Mixed Compliance Demonstrations?

- arxiv: <https://arxiv.org/abs/2606.20508>
- summary: This study investigates how different types of compliance demonstrations affect harmful compliance in language models, revealing that benign and harmful demonstrations interact in complex ways depending on the model and training methodology.
- methods: hypothesis testing, empirical analysis
- findings: Benign demonstrations can either reduce or increase harmful compliance depending on the model.; Preference optimization is critical in preventing benign demonstrations from increasing harmful compliance.; Demonstration ordering exhibits strong recency bias.; Models differ in how refusal interacts with in-context learning.

### Contagion Networks: Evaluator Bias Propagation in Multi-Agent LLM Systems

- arxiv: <https://arxiv.org/abs/2606.20493>
- summary: This study introduces Contagion Networks to measure how biases from large language model evaluators propagate in multi-agent systems, revealing significant contagion effects and proposing strategies for mitigation.
- methods: controlled 3-agent experiment, Cross-Agent Contagion Matrix measurement, spectral radius analysis
- findings: Evaluator biases propagate consistently between agents with gamma in [0.157, 0.352].; Homogeneous-model agents produce contagion coefficients 3-5x weaker than cross-model coefficients.; Increasing evaluator committee size from k=1 to k=3 reduces effective contagion by 72.4%.

### Fast Human Attention Prediction for Fixation-guided Active Perception in Autonomous Navigation

- arxiv: <https://arxiv.org/abs/2606.20491>
- summary: This study introduces GazeLNN, a lightweight scanpath prediction model that enhances robot autonomy by efficiently modeling human visual attention.
- methods: scanpath prediction, auto-regressive architecture, feature extraction, Reinforcement Learning
- findings: GazeLNN requires only 0.61 GFLOPs.; Achieves a 0.47 ScanMatch score on the MIT Low Resolution dataset.; Outperforms existing recurrent baselines across diverse evaluation metrics.; Reduces computational costs by 99.40%.; Accelerates inference by up to six times.; Enables human-fixation-guided perception during autonomous navigation.

### Beyond Global Replanning: Hierarchical Recovery for Cross-Device Agent Systems

- arxiv: <https://arxiv.org/abs/2606.20487>
- summary: The study introduces H-RePlan, a hierarchical replanning framework for multi-device agents that improves recovery from failures by separating device-local strategy recovery from global replanning.
- methods: hierarchical replanning, fault injection, cross-device workflows
- findings: H-RePlan outperforms single-strategy and coarse-grained multi-device baselines; achieves higher completion, instruction adherence, and perfect-pass rates; reduces token cost for reliable end-to-end success

### Optimal Order of Multi-Agent and General Many-Body Systems

- arxiv: <https://arxiv.org/abs/2606.20485>
- summary: This paper presents a framework for analyzing multi-agent systems with feedback loops, focusing on agent power and response functions to derive macroscopic properties and optimize collective behavior.
- methods: framework development, macroscopic property derivation, system-level utility function analysis
- findings: Stronger synchronization can increase collective output but may also increase systemic fragility and reduce mobility.; Order, entropy, information, and useful energy are task-dependent and system-relative concepts.; Agent power distributions and response functions can be measured and designed to optimize collective behavior.

### Your Mouse and Eyes Secretly Leak Your Preference: LLM Alignment using Implicit Feedback from Users

- arxiv: <https://arxiv.org/abs/2606.20482>
- summary: This study introduces the IFLLM dataset to leverage implicit human feedback for improving the alignment of Large Language Models (LLMs).
- methods: reward model, DPO
- findings: Implicit user feedback boosts the accuracy of the text-based reward model from 55% to 64%.; The relative response quality improvements nearly triple after applying DPO to eight LLMs.; Users exhibit diverse types of gazing behavior and mouse trajectories.

### GroundControl: Anticipating Navigation Failures in Vision-Language Agents via Trajectory-Consistent Uncertainty Estimates

- arxiv: <https://arxiv.org/abs/2606.20479>
- summary: The study introduces GroundControl, a trajectory-consistent uncertainty estimator that improves the reliability of vision-language navigation agents by anticipating failure dynamics during execution.
- methods: constant-velocity Kalman filter, normalized innovation statistics, trajectory features, Selective Risk--Coverage Navigation (SRCN), risk--coverage curves, AURC, E-AURC
- findings: Trajectory-consistent uncertainty achieves near-oracle ordering under success-based selective risk.; GroundControl outperforms entropy-, conformal-, and heuristic baselines.; GroundControl consistently achieves the lowest AURC and E-AURC across models and navigation splits.

### Marginal Advantage Accumulation for Memory-Driven Agent Self-Evolution

- arxiv: <https://arxiv.org/abs/2606.20475>
- summary: This paper introduces Marginal Advantage Accumulation (MAA) to improve batch-style trace distillation by addressing cross-batch feedback inconsistencies and achieving superior performance across multiple benchmarks and models.
- methods: Marginal Advantage Accumulation, EMA, semantic identity merging
- findings: MAA achieves the best results in 14 out of 16 settings; MAA consistently outperforms existing batch-level distillation baselines; MAA matches or surpasses online alternatives in most settings; MAA reduces optimization-phase token consumption by approximately 75%

### UltraQuant: 4-bit KV Caching for Context-Heavy Agents

- arxiv: <https://arxiv.org/abs/2606.20474>
- summary: This study investigates 4-bit KV-cache compression for context-heavy agents, focusing on design choices and optimizations that enhance performance on AMD GPUs.
- methods: codebook quantization, Walsh-Hadamard rotation, QJL removal, block-scale variants, optimized decode-attention kernels, UltraQuant
- findings: UltraQuant cuts P50 time-to-first-token by 3.47x in late rounds and 2.3x across all rounds; UltraQuant raises output throughput by 1.63x over the FP8 KV baseline

### Analyzing Defensive Misdirection Against Model-Guided Automated Attacks on Agentic AI Systems

- arxiv: <https://arxiv.org/abs/2606.20470>
- summary: This work analyzes the attack-defense dynamics in agentic AI systems using a probabilistic model and introduces a strategy called Contextual Misdirection via Progressive Engagement (CMPE) to mitigate prompt-injection and jailbreak attacks.
- methods: probabilistic modeling, detect-and-misdirect strategy, Contextual Misdirection via Progressive Engagement (CMPE)
- findings: Conventional detect-and-block defenses can lead to high attacker success rates as query budgets grow.; Detect-and-misdirect can reduce the positive predictive value of attacker-selected candidates.; CMPE reduces estimated attacker success rate upper bounds by up to two orders of magnitude.
