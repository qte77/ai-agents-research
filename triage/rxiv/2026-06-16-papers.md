## Extracts

### Neither Parallel Nor Sequential: How DiffusionGemma Actually Commits Tokens

- arxiv: <https://arxiv.org/abs/2606.14620>
- summary: This study investigates the decoding order of the DiffusionGemma 26B model, revealing that its token commitment is neither parallel nor block-autoregressive, but rather exhibits a partial left-to-right commit bias influenced by analysis granularity.
- methods: instrumentation, sampling, analysis of commit order
- findings: Decoding is neither parallel nor block-autoregressive.; Commit order is weak token by token but strengthens with coarser analysis.; Model commits in large simultaneous batches with undefined within-batch order.; Commitment is aggressive, finishing in a late burst within the step budget.; Task accuracy matches the autoregressive Gemma-4 sibling.

### Safe Reinforcement Learning of Autonomous Highway Driving: A Unified Framework for Safety and Efficiency

- arxiv: <https://arxiv.org/abs/2606.14609>
- summary: The study proposes a unified safe reinforcement learning framework, MoE-RM-SRL, that enhances safety and efficiency in decision-making for autonomous vehicles by integrating safe distance, reward machines, and mixture-of-experts.
- methods: safe distance, reward machines, mixture-of-experts, deep Q-networks, sparsely gated MoE layer
- findings: MoE-RM-SRL improves safety and efficiency over state-of-the-art baselines.; The framework enables safe and reliable behavior without sacrificing efficiency.; The architecture mitigates instability and impulsive transients in control.

### LoSoNA: A Benchmark for Local Social Norm Adaptation in Group Conversations

- arxiv: <https://arxiv.org/abs/2606.14600>
- summary: This study introduces LoSoNA, a benchmark for evaluating LLM-based agents' ability to adapt to local social norms in online group chats.
- methods: benchmarking, prompting conditions, evaluation of models
- findings: Naive prompting remains limited for most models; Explicit norm-aware prompting helps unevenly; Gemini 3.1 Pro reached 84.2% accuracy; Claude Fable 5 reached 81.6% accuracy

### When Errors Become Narratives: A Longitudinal Taxonomy of Silent Failures in a Production LLM Agent Runtime

- arxiv: <https://arxiv.org/abs/2606.14589>
- summary: This study investigates silent failures in a long-lived personal-assistant LLM agent system, identifying a unique failure pattern and proposing a taxonomy and defense framework.
- methods: longitudinal study, root-cause postmortems, retrospective audit
- findings: 70% of silent failures were caught by human user-view observation, not tests or audits; 0% ex-ante prevention but 87% regression blocking found in retrospective audit of incidents; incident latency tracks failure mechanism, not code complexity

### CARE: Controlling LLM-Generated Policies through Auditable Review of Evidence in Scientific Experimentation

- arxiv: <https://arxiv.org/abs/2606.14581>
- summary: The study introduces CARE, an auditable controller for optimizing high-throughput experimentation that balances LLM creativity with safety and stability.
- methods: auditable controller, public-evidence intervention gate, challenger ranking policies
- findings: CARE outperforms all other evaluated methods on Minerva/Olympus and ChemLex benchmarks; Final-best improved from 80.0 to 88.5 on Minerva/Olympus; Final-best improved from 83.9 to 92.1 on ChemLex; LLM self-evolution is more reliable under an auditable controller

### SIMMER: Benchmarking Latent Failures in LLM Executable Planning with a World Model

- arxiv: <https://arxiv.org/abs/2606.14574>
- summary: The study introduces SIMMER, a benchmark for evaluating latent failures in LLM planning in household environments, revealing significant shortcomings in current models and proposing a method to reduce these failures.
- methods: state machine executor, counterfactual foresight simulation
- findings: Even frontier models achieve at most 17% error-free plans.; Up to 56% of plans contain latent failures.; The majority of latent failures lead to irreversible consequences.; Explicit state reasoning can reduce latent failures by up to 72% and irreversible cases by up to 75%.

### StreamMemBench: Streaming Evaluation of Agent Memory for Future-Oriented Assistance

- arxiv: <https://arxiv.org/abs/2606.14571>
- summary: The study introduces StreamMemBench, a streaming benchmark designed to evaluate how personal-agent memory utilizes stored information and user interactions for future assistance.
- methods: benchmarking, experiments
- findings: Current memory systems often fail to use observed evidence.; Feedback is not reliably turned into follow-up behavior.; Four metrics diagnose evidence recall, initial evidence use, feedback incorporation, and follow-up reuse.

### ORCA: A Platform for Open-Source Dexterity Research

- arxiv: <https://arxiv.org/abs/2606.14561>
- summary: This work introduces the orca learning stack, an open-source research framework that unifies various aspects of dexterous robot learning to facilitate research in manipulation tasks.
- methods: low-level control, simulation, teleoperation, hand retargeting
- findings: The orca stack integrates control, simulation, and teleoperation behind a single interface.; It allows dexterous hand researchers to use the same data, training, and evaluation pipelines as non-dexterous robot learning.; A complete end-to-end workflow was demonstrated, including expert demonstrations, training an autonomous policy, and evaluating the learned policy.

### Spatially Conditioned Diffusion Policy: Learning Precise and Robust Manipulation with a Single RGB Camera

- arxiv: <https://arxiv.org/abs/2606.14535>
- summary: The study presents Spatially Conditioned Diffusion Policy (SCDP), a diffusion-based visuomotor policy that enhances manipulation tasks using a single-camera setup by leveraging end-effector trajectories as visual attention anchors.
- methods: diffusion-based policy, visual encoder, spatial conditioning module
- findings: SCDP outperforms strong single-view baselines; SCDP achieves performance comparable to multi-camera baselines; demonstrates precise manipulation and robustness to visual distractors

### AERMANI-PLACE: Language Guided Object Placement with Aerial Manipulators

- arxiv: <https://arxiv.org/abs/2606.14531>
- summary: The AERMANI-PLACE framework enables language-guided object placement for aerial manipulators by generating visual markers from natural language instructions and grounding them in the physical environment.
- methods: image editing model, depth observations, placement trajectory generation
- findings: The method achieved an average success rate of 87% on the test set for inferring placement locations from language instructions.; The approach transferred effectively to real-world aerial manipulation with an average success rate of 72%.

### Code Correctness Signals in LLM Hidden States: Pre-Generation Probing and Repair Geometry

- arxiv: <https://arxiv.org/abs/2606.14530>
- summary: This study investigates the legibility of code correctness in the hidden states of the Qwen3-4B-Instruct-2507 model, revealing insights into the model's performance on code generation and repair tasks.
- methods: linear decoding, residualization, nested cross-validation
- findings: The correctness of the model's first-attempt code is linearly decodable from the prompt-final hidden state with a high AUC.; The hidden state shift from a failing attempt to its repair shows a statistically detectable contrastive direction, but this is influenced by repair-context covariates.

### BayLing-Duplex: Native Full-Duplex Speech Dialogue with a Single Autoregressive LLM

- arxiv: <https://arxiv.org/abs/2606.14528>
- summary: The paper introduces BayLing-Duplex, a native full-duplex SpeechLM that enhances real-time speech interaction in chatbots by eliminating the need for an external turn-taking module.
- methods: autoregressive LLM, fine-tuning, lightweight DPO stage
- findings: BayLing-Duplex achieves 92% turn-taking success and 100% interruption success on InstructS2S-Eval.; The speech-response score improved from 2.17 to 3.39 over Moshi.; BayLing-Duplex matches or surpasses its turn-based counterpart on various benchmarks.

### From Shield to Target: Denial-of-Service Attacks on LLM-Based Agent Guardrails

- arxiv: <https://arxiv.org/abs/2606.14517>
- summary: This study reveals a novel vulnerability in LLM-based guardrails against prompt injection and jailbreak attacks, demonstrating how crafted data can exploit reasoning loops to execute denial-of-service attacks.
- methods: beam-search optimization framework, natural-language payload crafting, mechanism-aware structural mutations
- findings: Attack generalizes across diverse guardrail architectures and safety templates.; Payloads optimized on a single open-source surrogate transfer to eight leading model backbones.; Achieved 13--63× token amplification.; In real-world deployments, revealed up to 148× latency amplification.; A single poisoned document can saturate shared guardrail infrastructures.
